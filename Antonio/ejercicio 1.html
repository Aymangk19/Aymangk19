HTML

HTML, siglas en inglés de HyperText Markup Language (‘lenguaje de marcado de hipertexto’), hace referencia al lenguaje de marcado para la

elaboración de páginas web, no confundir con un lenguje de progrmación. Es un estandar del World Wide Web

Consortium (W3C)

organización dedicada a la estandarización de casi todas la stecnologías ligadas a la web.

Primeras especificaciones de HTML

Tim Berners-Lee1 en 1991​ describe 18 elementos que incluyen el diseño inicial y relativamente simple de HTML.

Berners-Lee consideraba el HTML una ampliación de SGML, pero no fue formalmente reconocida como tal hasta la publicación a

mediados de 1993, por la IETF2), de una primera proposición para una especificación del HTML: el borrador del Hypertext

Markup Language de Berners-Lee y Dan Connolly.

Marcador HTML

El HTML se escribe en forma de «etiquetas», rodeadas por corchetes angulares (<,>,/). El HTML también puede describir, hasta

un cierto punto, la apariencia de un documento, y puede incluir o hacer referencia a un tipo de programa llamado script, el cual puede

afectar el comportamiento de navegadores web
